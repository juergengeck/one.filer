ONE Filer Windows Package - Quick Start
=======================================

1. INSTALL ONE FILER:
   - Go to the "installer" folder
   - Right-click "install-one-filer-ubuntu-fixed.bat"
   - Select "Run as administrator"
   - Follow the prompts

2. START ONE FILER:
   - Use the desktop shortcut "ONE Filer"
   - OR open Command Prompt and run:
     wsl -d Ubuntu -- bash -c 'cd ~/one.filer ; ./start.sh'

3. TEST THE INSTALLATION:
   - Double-click "test-windows-access.bat" in this folder
   - OR navigate to O:\onefiler in Windows Explorer

4. TROUBLESHOOTING:
   - If WSL2 was just enabled, restart your computer
   - Make sure virtualization is enabled in BIOS
   - Check that Ubuntu is running: wsl -l -v
   - View logs: wsl -d Ubuntu -- tail -f ~/one.filer/logs/filer.log

For detailed instructions, see INSTALL_AND_TEST.md
